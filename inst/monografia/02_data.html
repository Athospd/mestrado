1 The Data

1.1 Training/Testing data

Two main data sources with almost ready to use labels were gathered:

  • Xeno-Canto.org
  • Wikiaves.com.br

Those websites are the biggest repository of Brazilian bird species sounds. The table 1.1 show the counts of MP3 downloaded by species.

library(magrittr)
data.frame(Species = list.files("../../data-raw/wav_16khz/") %>%
             stringr::str_remove_all("[.0-9]|(wav)") %>%
             stringr::str_remove_all("-$")
) %>%
  dplyr::count(Species, name = "#mp3", sort = TRUE) %>%
  janitor::adorn_totals() %>%
  knitr::kable(caption = "MP3 files downloaded.") %>%
  kableExtra::kable_styling(font_size = 12) %>%
  kableExtra::row_spec(6, bold = TRUE)
